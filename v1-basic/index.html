AngularJS Weather Widget

Version 1: Basic

This initial version of the widget illustrates how a self-contained AngularJS widget can be added to an existing web page.

The only integration points are:

  • A link to the widget's stylesheet in the page head.
  • Some simple HTML that bootstraps the AngularJS widget at the required place on the page.
  • Two script tags to include the core AngularJS library and the widget code itself.
